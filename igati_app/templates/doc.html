Igati API Documentation

Register User

POST
/register/
Request Body (JSON)
{
  "firstName": "John",
  "lastName": "Doe",
  "email": "john@example.com",
  "password": "secret123",
  "phoneNumber": "0712345678"
}
Response (201 Created)
{
  "message": "Successfully registered"
}
Error (400 Bad Request)
{
  "message": "Email already exists"
}

Login User

POST
/login/
Request Body (JSON)
{
  "email": "john@example.com",
  "password": "secret123"
}
Response (200 OK)
{
  "message": "Successfully logged in",
  "token": "eyJhbGciOi..."
}
Error (401 Unauthorized)
{
  "message": "Invalid credentials"
}

Add Product

POST (Multipart)
/add_product/
Form Data
name, description, category, price, stock, image (file)
Response (201 Created)
{
  "message": "Product added successfully",
  "product_id": 12
}

Add to Cart

POST
/add_to_cart/
Request Body (JSON)
{
  "product_id": 1,
  "user_id": 10
}
Response
{
  "success": true,
  "message": "Product added to cart",
  "quantity": 2
}
